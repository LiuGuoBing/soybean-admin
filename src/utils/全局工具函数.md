# 全局工具函数

## 目录结构

```text
├── utils                  // 全局工具函数(纯函数，不含状态)
│   ├── auth               // 用户鉴权工具函数
│   ├── common             // 通用工具函数
│   ├── crypto             // 数据加密工具函数
│   ├── form               // 表单相关工具函数
│   ├── router             // 路由相关工具函数
│   ├── service            // 请求相关的工具函数
│   └── storage            // 存储相关工具函数
```

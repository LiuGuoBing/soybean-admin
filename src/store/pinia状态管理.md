# pinia状态管理

## 目录结构
```text
├── store                  //pinia状态管理
│   ├── modules            //状态管理划分的模块
│   │   ├── app            // app状态(页面重载、菜单折叠、项目配置的抽屉)
│   │   ├── route          // route状态(动态路由、菜单、路由缓存)
│   │   ├── tab            // tab状态(多页签、缓存页面的滚动位置)
│   │   └── theme          // theme状态(项目主题配置)
│   └── subscribe          // 订阅状态(状态变更后执行一些操作)
```

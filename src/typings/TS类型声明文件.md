# TS类型声明文件(*.d.ts)

## 目录结构
```text
├── typings              // TS类型声明文件(*.d.ts)
│   ├── api               // 请求接口返回的数据的类型声明
│   ├── business          // 业务相关的类型声明
│   ├── env               // 项目启动或打包配置相关的类型声明
│   ├── expose            // vue 的defineExpose导出的类型声明
│   ├── global            // 全局相关的类型声明
│   ├── package           // 地图包相关的类型声明
│   ├── route             // 权限路由相关的类型声明
│   ├── router            // 业务相关的类型声明
│   ├── system            // 系统相关的类型声明
│   └── utils             // 工具类型声明
```
